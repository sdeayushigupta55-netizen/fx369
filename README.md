# FX369 React Website

A premium broker-style React landing page inspired by the provided reference websites.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important

Update the login and signup URLs in `src/App.jsx`:

```js
const loginUrl = '#'
const signupUrl = '#'
```

Replace the text content as needed for the final client version.
